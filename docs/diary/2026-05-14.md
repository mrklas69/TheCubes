**(F) Ramp Y slope-aware (DD-52).** User: „Stromy na rampách (oak) visí ve vzduchu, přepočítej výšku paty stromu na svahu." Sez. 40 měl konstantní `y_top + 1.0` (= střed ramp diagonal); jitter pozice mimo střed = floating nebo zapuštěná. Implementace: `ramps[].slopeDir = { dx, dz }` (unit vector low→high) přiřazen v candidate creation (edge: `pick.dir.dx/dz`, corner: `c.ddx/ddz`, diagonal: `tdrampCorner.ddx/ddz`). `decorate()` rozšířen: `slopeT = (jitterX·dx + jitterZ·dz) / (|dx|+|dz|)`, `decY = y_top + 1.0 + slopeT`. Normalizace unifikuje edge (norm=1) + corner/diagonal (norm=2). Tradeoff: TDRAMP je diskontinuální (step shape, ne smooth slope) — bilinear formula je mild aproximace, sub-prah pro exact step model. User test: „Opraveno OK."
