### Diskuse

Konceptuální debata o vizi projektu. Klíčové body:

- **Vize:** meta-sandbox s živým OOP modelem. Hráč rozšiřuje model a scéna se zaplňuje. Cílovka: mládež, demonstrace "tvorba > konzumace".
- **Meta-model vs. hráčský model:** meta-model stavíme my společně; hráčský model tvoří uživatel.
- **Inspirace PocketStory:** pattern ENTITY + RELATION + RULE + EVENT + QUEST, data-driven entity, model/engine separation. Ale podle KISS zatím nepřejímáme nic preemptivně — jen CUBES.
- **Kritické momenty:** A4 (neviditelné kostky rozbíjejí pojem "kostka" → OBJECTS jako kořen), A8 (TIME globální, ne per-object tick — statický M1), A11 (CUBES = "cokoli s polohou", varianta I), Q10 (stack: jen frontend, Three.js, bez backendu).
