## Sezení 19 — DONE.md split + prefix metapromptů `@` → `%`

Krátké meta-sezení (žádný kód), čistě dokumentační hygiena.

### Diskuse

**(1.) DONE.md split.** TODO.md narostl na 240 řádků a 95 % byly `[x]`
položky — fakticky archiv, ne aktivní seznam úkolů. User požádal
o vytvoření `docs/DONE.md` a přesun hotových.

Strategie: per-item rozdělení. DONE.md dostal kompletní strukturu
(M1–M7, M8+ Hotové, Sezení 14–18 + jejich „Příště — Hotové" podsekce);
TODO.md zkrácen na ~25 řádků (jen `[ ]` z M8+ + aktivní „Sez. 18 →
19 Příště"). `~~strikethrough~~` DROP položky zařazeny do DONE
s anotací DD nebo cleanup důvodu, aby budoucí context (pro auditu /
review) měl odkaz na rozhodnutí.

**(2.) Prefix metapromptů.** User upozornil, že `@` v Claude Code
otevírá file-picker (UI rušení) → kolize. Diskuse:

- **Stačí jen velká písmena** (`BEGIN`, `AUDIT:CODE`)? AI
  oponovala — slova `AUDIT`, `DOCS`, `END` v Czech prose mohou
  natečeně vznikat jako běžný text („uprav DOCS", „udělej AUDIT
  funkce"), false-positive triggery hrozí. Sigil je potřeba.
- **Volné znaky:** `%`, `^`, `&`, `*`, `=`, `+`, `~`, `:`. Rezervované
  v Claude Code: `@` (file picker), `/` (slash commands / skills),
  `#` (memory write), `!` (bash mode).
- **Doporučení `%`:** snadno dosažitelné Shift+5, vizuálně
  výrazné, ničím neoperuje. **Klíčový bod:** nezasahuje do `AUDIT:CODE`
  syntaxu (dvojtečka jako oddělovač zůstává). `:` jako prefix by
  vyžadoval rename `AUDIT:CODE` → `AUDIT.CODE` (kolize).

User schválil `%` + scope „I globální memory napříč projekty".

**(3.) Bulk rename.** PowerShell skript: `[System.IO.File]::ReadAllText`
+ `WriteAllText` s `UTF8Encoding($false)` (no BOM kvůli českým
diakritikám), pevný `tokenMap` (7 tokenů: `%AUDIT:CODE`, `%AUDIT:DOCS`,
`%BEGIN`, `%END`, `%THINK`, `%DOCS`, `%CALIBRATE`), exclude pattern
`\\file-history\\|\\backups\\|\\node_modules\\|\\\.git\\|\\dist\\|\\build\\|\\\.next\\`.

Scope: `~/.claude/CLAUDE.md`, `~/.claude/PROMPTS.md`, všechna
`~/.claude/projects/*/memory/*.md`, všechny zdrojové projekty
v `C:/Users/mrkla/source/` (md + ts + tsx + js + jsx + html).

**Výsledek:** **138 souborů změněno z 1770 zkontrolovaných** napříč
7 projekty (TheCubes, PocketStory, Stickman, Voidspan, Pixelodynamics,
Isometric, JePEK) + globální config + 5 projektových memory adresářů.

Žádné zbývající `@(BEGIN|END|THINK|DOCS|AUDIT:CODE|AUDIT:DOCS|CALIBRATE)\b`
v `C:/Users/mrkla/source/` po sweep — verifikováno grepem.
