### Tick model

- **1 wall sekunda = 1 tick** (globální `TIME.tick`, dosud DD-04 neused — konečně získává konzumenta).
- `productionTick(dt)` v render loopu, ne per-1s setInterval. `dt = wall second elapsed × world.TIME_SCALE`. Clamping `dt ≤ 0.1 s` (proti tab-sleep spikes).
- Rate jako float ks/s (les 0.5/s = 30/min), nikoli integer per-tick.
- Resource unit MVP = abstraktní „ks" napříč (logs, water, …). Voidspan `formatScalar` (l/m³/kg) odložen na Phase 2.

### Event Log model

- Voidspan-derived: 4-znakové verbs (MVP: `PROD`, `DRN`, `PAUS`, `RSUM`).
- Ring buffer kapacita 100, viditelných 5 v `#eventlog` (left bottom).
- **Emit jen na state change**, ne per-frame:
  - `PROD` / `DRN` — jen na **floor crossing** (whole-unit produkce / spotřeba). Per-frame produkce 60× by zahltila log.
  - `PAUS` / `RSUM` — jen na change `facility.PAUSED` true↔false.
- Severity-colored CSS: PROD zelená, DRN neutrální šedá, PAUS amber, RSUM cyan.

## Kód

Boční větev `feat/factory` — fáze A implementována.

### Model (`src/model.js`)

- **`WORLD` rozšíření** — `WIND_STRENGTH` (zachováno), nové `TIME_SCALE` (default 1.0), `RESOURCES` (objekt 6 klíčů iniciovaný z `RESOURCES_DEF`, default 0).
- **Nová sekce FACTORY TOY** (~150 LOC) — registry + třídy:
  - `RESOURCES_DEF` — 6 surovin (`name_cs`/`name_en`/`category solid|fluid`/`unit "ks"`).
  - `RECIPES_DEF` — 2 recepty (`sawmill { logs:1 → planks:0.8, 1.0/s }`, `crusher { stone:1 → gravel:1.2, 0.8/s }`).
  - `FACILITY_DEF` — 7 KINDů (forest/quarry/well/coal_mine generátory, sawmill/crusher transformery, storage). Per-KIND: `type`, `outputs`/`recipe`, `buffer_capacity`, `color` (placeholder mesh), `name_cs`.
  - `FACILITY extends CUBES` — base třída, atributy `KIND`, `BUFFER` (`{resource_id: amount}`), `PAUSED`, `PAUSE_REASON`.
  - `GENERATOR`, `TRANSFORMER`, `STORAGE` extends FACILITY — sourozenci, dispatch v engine podle `FACILITY_DEF[KIND].type`.

### Engine (`src/main.js`)

- **Sekce FACTORY TOY ENGINE** (~200 LOC) před `createMeshFor`:
  - `facilities[]` registry (analog `animators[]`).
  - `events[]` ring buffer (cap 100), `EVENT_LOG_VISIBLE = 5`.
  - `logEvent(verb, target, detail)` + `renderEventLog()` — DOM patch posledních 5 do `#eventlog`.
  - `maybeLogProduction()` / `maybeLogConsumption()` — emit `PROD`/`DRN` jen na floor crossing per (facility, resource).
  - `setPaused(facility, paused, reason)` — emit `PAUS`/`RSUM` jen na state change.
  - `generatorTick(facility, def, dt)` — produkuje `outputs[r]*dt`, ořez na `buffer_capacity`, pauza na all-blocked.
  - `transformerTick(facility, def, dt)` — 3 fáze: input check (`chybí <r>`), output capacity check (`buffer <r> plný`), apply.
  - `productionTick(dt)` — iteruje `facilities[]`, dispatch dle `type`.
  - `aggregateResources()` — reset `world.RESOURCES[r] = 0`, sum přes `f.BUFFER[r]`, DOM update `#res-<r>` čítačů.
- **`createMeshFor`** — nová větev `instance instanceof FACILITY` před PATH. `createFacilityFor(instance)` — 1×1×1 `BoxGeometry` s `MeshStandardMaterial(color: FACILITY_DEF[KIND].color)`, snap-to-grid (DD-28 BLOCKS-style grid-center), registrace do `facilities[]`. Sdílená geometrie `_facilityGeom`, per-KIND materiál cache `_facilityMatCache`.
- **Render loop** — clamp `dt ≤ 0.1`, `simDt = dt × world.TIME_SCALE`; `productionTick(simDt)` + `aggregateResources()` před `updateBubbleTails`.
- **Test scéna v `buildScene`** — 3 fasility na grass podlaze (Y=0): `fac_forest_0` (-2,0,2) `KIND=forest`, `fac_sawmill_0` (0,0,2) `KIND=sawmill` s pre-stocked `BUFFER.logs = 50`, `fac_storage_0` (2,0,2) `KIND=storage`. Bez PATH (fáze A): sawmill čerpá z vlastního pre-stocked bufferu, fáze B nahradí PATH.
