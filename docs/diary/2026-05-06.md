### Schválená rozhodnutí

- **DD-22 — Pevné měřítko + Y konvence.** 1 TC voxel = 1 m, 1 MV voxel =
  1/16 TC = 6.25 cm, default `SCALE: 0.625`. Velikost objektu řídí počet

  mesh = gy + 0.5` kde gy je grid souřadnice voxelu, na kterém model leží.

  → vizuální jazyk je výhradně voxelový. Žádné Cylinder/Cone/Sphere/Torus/
  Icosahedron pro core gameplay. Pravidlo dispatche: vegetace/kameny/mraky/

  + KIND sub-buildery). Komplexní entity (vozidla, postavy, charakteristické
  landmarky) → externí VOXEL_MODEL. Voxelová podlaha → TCUBES. Dialog/UI →
  SPRITES (spec-case). Sez. 15 cleanup smazal 7 non-voxel tříd + Scéna 1 +
  scene switcher + LIT + balloon_bob + classic TREE varianta + helpery
  (~720 řádků).

- **DD-24 — Shape × Surface separation.** VOXEL_MODELy s jednolitým povrchem
  rozděleny na shape (1 MV soubor s abstract paletou 4 indexy) + surface
  (JSON paleta 4 RGBA). Pre-build skript generuje kombinace per disk.
  Pojmenování `<shape>-<surface>` (kebab-case). Nekvalifikuje pro vozidla/
  postavy/stromy (multi-color).
